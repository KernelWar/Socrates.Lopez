💻 KidCode

          

            KidCode es una aplicación de escritorio desarrollada en Java que actúa como un compilador de
            pseudocódigo con interfaz gráfica.
            Permite escribir, analizar y ejecutar programas mediante un editor de código integrado, consola de errores,
            tabla de tokens, generador de código intermedio y visualización del árbol sintáctico.
            Su objetivo es ofrecer una herramienta educativa para comprender los procesos internos de compilación.
            Proyecto open source bajo licencia MIT.
